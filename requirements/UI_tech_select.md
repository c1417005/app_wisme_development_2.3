# Implementation Tasks for Wisme Frontend

既存の Django モデルおよびバックエンドロジックと連携しながら、以下のUI実装を行ってください。

1. **ベーステンプレートの構築**
   - Tailwind CSS と Vue.js (CDN) を読み込み、ベースとなるオフホワイトのデザインを作成せよ。
   - `base.html` に Lucide Icons を導入せよ。

2. **動的エディタの実装**
   - メモ作成画面において、Vue.js を用いて「章（Chapter）」を動的に追加・削除できる UI を構築せよ。
   - 章ごとのデータと全体感想文を一つの JSON 構造で Django の `POST` リクエストへ送れるようにせよ。

3. **非同期単語検索と自動紐付け**
   - 検索窓の入力をトリガーに Axios で Django の検索ビューを叩き、結果をサイドバーに表示せよ。
   - 検索成功時、別の非同期リクエストを送って現在のメモ ID と単語 ID を紐付ける処理を実装せよ。
   - 検索中はスケルトン UI またはプログレスバーを表示せよ。

4. **フラッシュカードの構築**
   - 単語帳ページにおいて、カードをクリックすると「パタッ」と裏返るアニメーションを Tailwind CSS のユーティリティクラスで実装せよ。

5. **通知システム**
   - 保存ボタン押下時、トースト通知（Toast）を表示して「保存完了」を知らせる処理を Vue.js で実装せよ。

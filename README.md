# [Wisme]

[読書メモと自分だけの語彙集作成WEBアプリ]

## 🚀 概要 (Overview)
[このプロジェクトが解決する課題や、主な目的を2〜3行で書く]
私の趣味は読書です。平均月2～3冊読んでいます。
自分の
・「分からない言葉を調べてタブが乱立する。タブ整理がストレス。」
・「調べただけでは"点"の知識なためすぐ忘れてしまう。」
・「検索結果をいちいちどこかにメモするのも面倒」
という体験を踏まえ、同じく読書が趣味の方、自己成長のためビジネス書や自己啓発本を読まれる方も同じ思いなのではと考えました。

 このプロダクトにより読書メモ（得た知識・感想のアウトプット）と言葉の検索が1つのページで完結します。
 言葉の検索にはGemini APIを利用しています。分からない単語を入力するだけで自動で意味が検索され、結果が保存されます。
 シームレスな動作と、読書メモと結びつくことで知識が"線"になることで、読書体験をより価値のあるものにします。
 

 今後は音声入力等の機能を実装し、利便性の向上を図っていきます。

# 008 多言語対応（日本語・英語）の実装

この機能は機能拡張ということにし、まだ実装しない

## 1. 概要



### 目的
アプリケーションの UI を日本語・英語の 2 言語に対応させる。  
言語切り替えボタンを設置し、ユーザーが任意のタイミングで言語を変更できるようにする。

### 背景
- 将来的な海外ユーザーへの対応と、英語学習コンテキストでのアプリ利用を見越した対応。
- Django 標準の `i18n` フレームワーク（`gettext`）を使用することで、テンプレートとビューの両方に対応できる。

---

## 2. 実装方針

### 2-1. `settings.py` の変更

```python
from django.utils.translation import gettext_lazy as _

MIDDLEWARE += [
    'django.middleware.locale.LocaleMiddleware',
]

LANGUAGE_CODE = 'ja'
LANGUAGES = [
    ('ja', _('Japanese')),
    ('en', _('English')),
]
USE_I18N = True
LOCALE_PATHS = [BASE_DIR / 'locale']
```

### 2-2. URL の追加

`myproject/urls.py` に `i18n_patterns` を適用：

```python
from django.conf.urls.i18n import i18n_patterns
import django.conf.urls.i18n as i18n

urlpatterns += [
    path('i18n/', include('django.conf.urls.i18n')),
]
```

### 2-3. テンプレートの翻訳マーキング

```html
{% load i18n %}
<h1>{% trans "読書メモ一覧" %}</h1>
<button>{% trans "新規作成" %}</button>
```

- 全テンプレートの表示文字列を `{% trans "..." %}` でマークする。
- Python コード内の文字列は `gettext_lazy` を使用する。

### 2-4. 翻訳ファイルの生成

```bash
python manage.py makemessages -l en
# locale/en/LC_MESSAGES/django.po を編集して翻訳を追加
python manage.py compilemessages
```

### 2-5. 言語切り替えボタンの設置

`templates/base/wisme_base.html` に追加：

```html
{% load i18n %}
<form action="{% url 'set_language' %}" method="post">
  {% csrf_token %}
  <input name="next" type="hidden" value="{{ request.get_full_path }}">
  <select name="language" onchange="this.form.submit()">
    {% get_available_languages as LANGUAGES %}
    {% for lang_code, lang_name in LANGUAGES %}
      <option value="{{ lang_code }}" {% if lang_code == LANGUAGE_CODE %}selected{% endif %}>
        {{ lang_name }}
      </option>
    {% endfor %}
  </select>
</form>
```

## 3. 影響範囲

| ファイル | 変更内容 |
|---|---|
| `myproject/settings.py` | `LocaleMiddleware`・`LANGUAGES`・`LOCALE_PATHS` を追加 |
| `myproject/urls.py` | `i18n` URL を追加 |
| `templates/**/*.html` | 全表示文字列に `{% trans %}` を追加 |
| `locale/en/LC_MESSAGES/django.po` | 英語翻訳ファイルを新規作成 |
| `locale/ja/LC_MESSAGES/django.po` | 日本語翻訳ファイルを新規作成（必要に応じて） |
